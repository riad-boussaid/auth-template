# Auth-Template

Integration of Hono RPC + TanStack Query and Next.js...

## Prerequisites

**Node version 20.15.x**

## Tech Stack

- Next.js
- ShadcnUi
- TailwindCSS
- Drizzle ORM
- Hono RPC
- TanStack Query
- React Hook Form
- Zod

## Install packages

```shell
npm i
pnpm i
bun i
```

## Setup .env file

```shell
DATABASE_URL = "";

ENCRYPTION_KEY;

GMAIL_USER = "";
GMAIL_PASSWORD = "";

GOOGLE_CLIENT_ID = "";
GOOGLE_CLIENT_SECRET = "";

FACEBOOK_CLIENT_ID = "";
FACEBOOK_CLIENT_SECRET = "";

NEXT_PUBLIC_APP_URL = "";

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME = "";
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET = "";
CLOUDINARY_API_KEY = "";
CLOUDINARY_API_SECRET = "";
```

### Start the app

```shell
npm run dev
pnpm dev
bun dev
```
